JavaScript Functions

This is a webpage to practice creating and using JavaScript logic functions. Please click on any of the buttons below to see how they work!

1. Sum of your input:

add a number and this function will output the sum of the number 1 through n.

2. Product of your input:

add a number and this function will output the product of the number 1 through n.

3. Is your number divisible by three or five:

add a number and this function will see if n is divisible by three or five.

4. Print all prime numbers:

Input up to how many numbers you would like to print.

5. Guess a number:

input a number and this function will tell you if you're above, below or just right. Afterwords, it will tell you the amount of guesses you made.
